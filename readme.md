> Build E-Shop with Admin Panel using the technologies: Nodejs , Mongo, Angular 12, Nrwl NX Monorepo, PrimeNG

### Get started

```shell script
git clone
```
```shell script
cd backend
```
```shell script
npm i
```
```shell script
npm run dev
```
```shell script
cd frontend
```
```shell script
npm i
```
```shell script
ng serve ngshop -o
```
```shell script
ng serve admin -o
```
